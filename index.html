What is Clo Clo?

Clo Clo is a cloud service that provides an easy way to send, receive and interact with data from any Internet enabled device.

Features

  • RESTful API provides a structured and simple way to interact with Clo Clo data streams.
  • Built on a custom time series storage system the merges the speed of Redis with the scalability of MySQL.
  • An easy to use dashboard that allows you to manage your streams.
Checkout the project here Github

Getting Started

Push Data
PUT http://www.clo-clo.net/streams/[stream_id]
arguments: 
key=[your_api_key]
arg1=value
arg2=value
Pull Data
GET http://www.clo-clo.net/streams/[stream_id] 
optional arguments:
start=[start time in epoch]
end=[end time in epoch]
key=[your_api_key] if your stream is not public
